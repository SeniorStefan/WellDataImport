WPF-приложение на C# /.NET 4.7.2, которое помогает загрузить, проверить и собрать сводную информацию из CSV-файлов со скважинами.

Что делает программа

Читает CSV Формат: WellId;X;Y;DepthFrom;DepthTo;Rock;Porosity Пример строки: A-001;82.10;55.20;0;10;Sandstone;0.18

Валидация

DepthFrom < DepthTo и ≥ 0
Porosity в [0 .. 1]
Rock не пустой
Интервалы одной скважины не пересекаются Ошибки собираются в отдельную таблицу (номер строки, WellId, описание).
Сводка по скважине
Общая глубина — максимальный DepthTo
Кол-во интервалов
Средняя пористость, взвешенная по толщине интервалов
Самая распространённая порода (по сумме толщин)
Экспорт итоговой сводки в JSON.
Архитектура (коротко)

Папка/Класс	Что хранит
Models	Well, Interval, WellSummary, ValidationError
Services	WellDataService — чтение CSV, валидация, построение сводок
ViewModels	MainViewModel — логика UI; RelayCommand — поддержка команд
UI	MainWindow.xaml — тёмная тема, три таблицы: сводка, интервалы, ошибки
Как запустить

Откройте WellDataImport.slnx в VS 2019 или новее.
Убедитесь, что установлен .NET Framework 4.7.2.
Восстановите NuGet-пакеты (Restore NuGet Packages).
Сборка → Debug, Rebuild Solution.
Запустите (F5 или Ctrl+F5).
Как работать в приложении

Нажмите «Загрузить CSV», выберите файл.
После загрузки слева-вверх увидите сводку по скважинам, слева-вниз — интервалы выбранной скважины, справа — список найденных ошибок.
Чтобы сохранить результаты, нажмите «Экспорт JSON» и укажите путь.
