WPF-приложение на C# /.NET 4.7.2, которое помогает загрузить, проверить и собрать сводную информацию из CSV-файлов со скважинами.

--------------------------------------------------------------------

Что делает программа

1. Читает CSV
Формат:
WellId;X;Y;DepthFrom;DepthTo;Rock;Porosity
Пример строки: A-001;82.10;55.20;0;10;Sandstone;0.18

2. Валидация
* DepthFrom < DepthTo и ≥ 0
* Porosity в [0 .. 1]
* Rock не пустой
* Интервалы одной скважины не пересекаются
Ошибки собираются в отдельную таблицу (номер строки, WellId, описание).

3. Сводка по скважине
* Общая глубина — максимальный DepthTo
* Кол-во интервалов
* Средняя пористость, взвешенная по толщине интервалов
* Самая распространённая порода (по сумме толщин)

4. Экспорт итоговой сводки в JSON.

--------------------------------------------------------------------

Архитектура (коротко)

| Папка/Класс | Что хранит                                                            |
| ----------- | --------------------------------------------------------------------- |
| Models      | Well, Interval, WellSummary, ValidationError                          |
| Services    | WellDataService — чтение CSV, валидация, построение сводок            |
| ViewModels  | MainViewModel — логика UI; RelayCommand — поддержка команд            |
| UI          | MainWindow.xaml — тёмная тема, три таблицы: сводка, интервалы, ошибки |

--------------------------------------------------------------------

Как запустить

1. Откройте WellDataImport.slnx в VS 2019 или новее.
2. Убедитесь, что установлен .NET Framework 4.7.2.
3. Восстановите NuGet-пакеты (Restore NuGet Packages).
4. Сборка → Debug, Rebuild Solution.
5. Запустите (F5 или Ctrl+F5).

--------------------------------------------------------------------

Как работать в приложении

1. Нажмите «Загрузить CSV», выберите файл.
2. После загрузки слева-вверх увидите сводку по скважинам, слева-вниз — интервалы выбранной скважины, справа — список найденных ошибок.
3. Чтобы сохранить результаты, нажмите «Экспорт JSON» и укажите путь.
